# Project Memory

Internal milestone timeline board built as a local-first Next.js app. Rows represent sections or projects, time runs horizontally, and milestones are automatically placed by date inside the selected section.

Current status: `V1 in active UI/UX refinement`.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- `date-fns`
- Browser `localStorage` for persistence

- Keep entries concise and dated with `YYYY-MM-DD`.
- Use this folder as the durable memory for the project instead of relying on chat history.
